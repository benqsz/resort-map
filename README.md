# Resort map

## Prerequisites
- Nodejs
- pnpm

## Starting the app

```bash
  # install deps
  pnpm install
  
  # start app with normal Next.js dev server
  pnpm dev
  
  # or start with CLI args 
  node run.js -- --map path/to/map.ascii --bookings path/to/bookings.json
```

## Testing
```bash
  pnpm test
```
